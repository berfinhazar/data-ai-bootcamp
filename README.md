## Project Overview

This project analyzes a large-scale Hacker News dataset containing approximately **876,000 records** to uncover how the technology community has evolved over time.

Rather than focusing solely on descriptive statistics, we built an end-to-end analytics and machine learning pipeline that explores:

*  The rise and fall of major technology trends such as **Artificial Intelligence, Cryptocurrency, and Cloud Computing**
*  Community sentiment shifts and emotional reactions across different periods
*  Fear, uncertainty, and panic signals reflected in discussion titles
*  Technology hype cycles and their long-term trajectories
*  Predictive models capable of identifying patterns within Hacker News content

The result is a data-driven "time capsule" of the tech industry, combining large-scale data analysis, natural language processing (NLP), visualization, and machine learning to reveal how conversations, concerns, and innovations have changed throughout the years.
